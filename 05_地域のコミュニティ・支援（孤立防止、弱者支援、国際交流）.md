# 4. 地域のコミュニティ・支援（孤立防止、弱者支援、国際交流）

## 主要論点

- 地域内のコミュニケーションが少ない
- 障害者・児、難病患者、非課税世帯などの弱者が生きづらい社会になっている
- グローバル化する社会で外国人とのコミュニケーションがなかなか取れない

## 課題① 地域内のコミュニケーションが少ない

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・隣近所とのコミュニケーションを今以上に取る<br>・家族や職場以外の人と交流するようにする |
| 地域の取り組み | ・自治会の運営を確立する |
| 行政の取り組み | ・各自治会の現状を把握する<br>・自治会の支援の強化（以前に戻す）<br>・一時預かりでなく、日常生活用具、補装具の公費負担強化<br>・孤立防止のため、趣味などの共通点がある人向けのイベントを行う |
| その他の主体の取り組み | ・コミュニティの強化のため、企業のキャラクターを利用して、広告などで挨拶の大切さをPRする<br>・企業主催で、地域との交流会を行う |

## 課題② 障害者・児、難病患者、非課税世帯などの弱者が生きづらい社会になっている

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・一人一人、何が問題か声を上げる |
| 地域の取り組み | ・講習会、家族会、相談会、集いなどを開催したり、声掛けをする |
| 行政の取り組み | ・医療機関の選択の自由の権利や、公平で良質な医療を受ける権利を守る<br>・家族会などの相談できる集いを多数開催する |

> その他の意見
> - 本人も家族も就業困難で収入もない中、医療費などが高額なため我慢するしかない。入院も隣の大阪市の方が支援が厚く差がある。同じ気持ちの患者と家族同士で悩みを打ち明ける場所が必要。

## 課題③ グローバル化する社会で外国人とのコミュニケーションがなかなか取れない

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・積極的に外国人に話しかけて行く |
| 地域の取り組み | ・地域に在住している外国人の方と交流を図る |
| 行政の取り組み | ・東大阪国際交流フェスティバルを西地区だけでなく中・東地区でも開催できるようにする<br>・もっと広く学生・社会人にまで広げた留学制度を設ける |
| その他の主体の取り組み | ・ライオンズクラブか商工会議所の社会人留学助成制度などを、積極的に広報する |
