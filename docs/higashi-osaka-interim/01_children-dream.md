# 1. 子どもの夢と、それを支える環境

- 夢を持った時にその職業などと触れ合える機会が少ない
- 夢を持っていない人について、さらに考える必要がある

## 課題① 夢を持った時にその職業などと触れ合える機会が少ない

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・自分でアウトプットの場、需要を探す<br>・本を読んだりして、知識を身につける<br>・職業だけじゃなく、やりたい事をはっきりさせる<br>・職業体験などできるものがないか調べる |
| 地域の取り組み | ・職業体験などをできる環境を提供する<br>・プチ依頼の枠組みをつくる（子供がプレゼン→依頼→受注）<br>・地域の人がどんな職業をしているのかを広めるポスターなどを作る |
| 行政の取り組み | ・学校でいろんな職業を体験できるイベントを開催するなど、職業に触れる機会や体験できる機会を増やす<br>・子どもたちへの色々な賞を増やしす（その後の仕事につながることもあり得る) |
| その他の主体の取り組み | ・その職業の団体が、子どもたちと触れ合えるような企画をする<br>・企業が、簡易職業チャレンジブースなどを設置する |

## 課題② 夢を持っていない人について、さらに考える必要がある

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・必ずしも夢を持つ必要はないと思う<br>・将来の夢は「職業」ではなく、職業は手段で、その目的はまた別であることを考えるようにする<br>・読書やネット情報、博物館に行ったり自分に刺激を与える<br>・夢の実現に向けて、子どもはその一歩がわからない時が多いので、背中を押してあげられるようにする |
| 地域の取り組み | ・夢ができた時に実現できる環境、夢を見つける環境を、地域として少しでも整えていく |
| 行政の取り組み | ・「多様な体験」や「挑戦を許容する文化」をつくることのできる仕組みづくりを行う<br>・子どもたちが興味を持つように、多くの職業の情報を発信する<br>・市のイベント内で簡易職業チャレンジブースの設置や、博物館での広報活動などに力を入れる<br>・新たな夢の創出や、夢ができた時に実現に向けた心を育むために、若者がチャレンジできる場を東大阪市が提供する |
