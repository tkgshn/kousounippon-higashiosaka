# 2. 子どもの教育・学校・居場所

## 主要論点

- 学校に行きづらい子どもが増えている

## 課題① 学校に行きづらい子どもが増えている

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・不登校の理由を根掘り葉掘り聞かず、まず「受容」するよう心掛ける |
| 行政の取り組み | ・教室以外の居場所（保健室登校や別室登校）を機能させていく<br>・現役を引退した栄養士やシェフにメニュー作成などの協力をお願いする<br>・東大阪市でも「学びの多様な学校」（不登校特例校）の設置を検討する。特例校への補助も併せて検討する（足立区の東京みらい中学校やN高等学校（通信制高校）のように、従来の学校以外の選択肢が増えることが、子どもの可能性を広げる上で重要） |
| その他の主体の取り組み | ・給食を作る事業者自身がアンケートをとり、子どもたちの意見を聞く |
