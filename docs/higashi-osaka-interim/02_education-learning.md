# 子どもの教育・学校・居場所

## 課題1: 学校に行きづらい子どもが増えている

### 個人の取り組み
- 不登校の理由を詮索し過ぎず、まず受容する姿勢を取る。

### 地域の取り組み
- （現時点で具体的な記載なし）

### 行政の取り組み
- 教室以外の居場所（保健室登校や別室登校）を機能させる。
- 引退した栄養士やシェフに協力を依頼し、食の面から支援を強化する。
- 東大阪市でも「学びの多様な学校」（不登校特例校）の設置を検討する。
- 特例校など従来の学校以外の選択肢を広げるための補助を検討する。

### その他の主体の取り組み
- 給食事業者がアンケートで子どもの意見を集め、改善に活かす。
