# 3. 医療・生活環境・治安（街灯・道路）

- 市内の救急病院に受け入れを断られることが多い
- 夜道が暗いことへの安全面の不安
- 道路がガタガタ、白線が薄くて危険、歩道が少ない・狭い」ため自転車と歩行者が混在し危険
- 遊べる場所（商業施設、娯楽施設）が市内に少ない

## 課題① 市内の救急病院に受け入れを断られることが多い

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・救急車を使わない時は、近くのいくつかの病院を把握し、色々な選択肢を持っておく<br>・救急の仕組みを理解する |
| 地域の取り組み | ・救急医療の仕組みや近くの病院の特性などについて、地域内で共有する |
| 行政の取り組み | ・コールセンターを設置し、電話内容の記録を取る 。より良いものにするために、翌日、その時の対応を検証する（電話対応は外部委託を行う）<br>・病院の受け入れ体制（救急車でなければダメ、という運用）の改善を働きかける |

## 課題② 夜道が暗いことへの安全面の不安

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・自宅前や自分の事業所前をライトアップする |
| 地域の取り組み | ・自宅前や事業所前のライトアップに対する支援、補助、盛り上げ、風潮づくりを行う |
| 行政の取り組み | ・夜でも明るいキラキラのまちにするため、イルミネーションを増やすなどライトアップする |
| その他の主体の取り組み | ・商工会議所など主催して賞の創設を考える(キラキラで表彰) |

## 課題③ 道路がガタガタ、白線が薄くて危険、歩道が少ない・狭いため自転車と歩行者が混在し危険

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・どこが一番歩道が少ないかなど、市内の道路状況を調べる |
| 地域の取り組み | ・優先順位も含めて改善できないか、役所などに相談してみる |
| 行政の取り組み | ・歩道整備を順次進める。また、市の歩道など道路状況についての意識調査をする |

## 課題④ 遊べる場所（商業施設、娯楽施設）が市内に少ない

| 区分 | 取り組み案 |
| --- | --- |
| 個人の取り組み | ・東大阪内のあまり知られていない遊び場を調べる |
| 地域の取り組み | ・地域の中の遊び場のマップを作る<br>・子ども会のイベントや地域の行事などを積極的に掲示板貼り出す |
| 行政の取り組み | ・空いてる土地に遊び場の設置の検討<br>・ドリーム21などで中高生向けのイベントをもっと開催する |
| その他の主体の取り組み | ・企業主催のイベントを、もっと東大阪で行う |
