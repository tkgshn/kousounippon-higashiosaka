# 子どもの夢と、それを支える環境

## 課題1: 夢を持った時にその職業などと触れ合える機会が少ない

### 個人の取り組み
- 自分でアウトプットの場や需要を探す。
- 本を読んだり学んだりして、興味のある分野の知識を身につける。
- 職業という目標だけでなく、やりたいこと自体を明確にする。
- 職業体験などの機会を調べ、積極的に参加する。

### 地域の取り組み
- 職業体験を提供できる環境や仕組みをつくる。
- 子どもによるプレゼン→依頼→受注のような「プチ依頼」の枠組みを整える。
- 地域の人がどのような職業に就いているかを紹介するポスターなどを作成する。

### 行政の取り組み
- 学校で多様な職業体験イベントを開催し、職業に触れる機会を増やす。
- 子ども向けの賞を拡充し、受賞が将来の仕事につながる可能性を広げる。

### その他の主体の取り組み
- 各職業団体が子どもと触れ合える企画を実施する。
- 企業が簡易職業チャレンジブースなどを設置し、職業体験の場を提供する。

## 課題2: 夢を持っていない人について、さらに考える必要がある

### 個人の取り組み
- 必ずしも「夢＝職業」である必要はないという視点を持つ。
- 職業は目的を実現する手段であると捉え直し、自分の目的を考える。
- 読書や博物館など新しい刺激を得られる場に触れて視野を広げる。
- 夢の実現に向けた最初の一歩が分からない子どもを励まし、背中を押す。

### 地域の取り組み
- 夢を見つけ、実現に向けて動ける環境を地域として整える。

### 行政の取り組み
- 多様な体験や挑戦を許容する文化を育む仕組みを整備する。
- 子どもが興味を持てるよう、多くの職業情報を継続的に発信する。
- 市のイベントに簡易職業チャレンジブースを設置するなど PR を強化する。
- 若者が新たな夢に挑戦できる場を提供し、チャレンジを支援する。
